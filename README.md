# Intake — Universal Capture for macOS

One shortcut to capture anything. Route thoughts, tasks, and bookmarks to 20+ apps instantly.

**[Download Latest Release](https://github.com/felciano/intake-app-releases/releases/latest)** | **[Landing Page](https://felciano.github.io/intake-app-releases)**

---

## Features

- **One Shortcut** — Press `Cmd+Shift+Space` from anywhere to capture
- **20+ Destinations** — Obsidian, Things, Todoist, Apple Reminders, Raindrop.io, NotePlan, Drafts, Day One, Bear, Craft, OmniFocus, TickTick, Due, files, webhooks, and more
- **Smart Routing** — Set a default or pick a destination with keyboard shortcuts
- **Privacy First** — All data stays on your Mac; API keys stored in Keychain
- **Capture History** — Browse, search, and filter past captures
- **Auto-Updates** — New versions delivered and verified automatically

## Install

1. Download the latest DMG from [Releases](https://github.com/felciano/intake-app-releases/releases/latest)
2. Open the DMG and drag Intake to Applications
3. Launch Intake — grant Accessibility permission when prompted
4. Configure your destinations in Settings

## System Requirements

- macOS 13.0 (Ventura) or later
- Apple Silicon or Intel Mac
